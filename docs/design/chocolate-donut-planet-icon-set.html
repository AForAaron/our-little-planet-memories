纯巧克力甜甜圈星球图标

去掉草莓后，图标会更克制、更高级，也更不容易偏向甜品店。这里保留“甜甜圈星球 + 奶巧克力淋面 + 轨道”的核心记忆点，适合你现在温柔、私密、有宇宙感的页面。

01 图标变体

建议主站使用第一款。第二款适合登录页，第三款适合水印和空状态。

A · 主品牌版

最推荐。奶油甜甜圈、巧克力淋面、前后轨道都完整，识别度足够，但没有草莓那么甜。

B · 奶油浅底版

更柔和，适合登录页、注册页、设置页。容器不抢眼，图标像一枚小徽章。

C · 线性水印版

适合照片占位、空状态、页面角落。只保留轮廓和巧克力语义，存在感最低。
